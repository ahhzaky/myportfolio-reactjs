## On Docker

Pull this images

```
docker pull ahmadzky08/myportfolio-react
```

AND RUNN

```
docker run -it -p 8888:3000 ahmadzky08/myportfolio-react
```

AND CHECK RUN ON LOCALHOST

```
http://localhost:8888/
```
